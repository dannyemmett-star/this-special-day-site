# This Special Day Website

Public website for This Special Day, a personalized birthday magazine experience for businesses.

## Website Files

- `index.html` - main homepage
- `styles.css` - website styling
- `contact.html` - contact page
- `start.html` - start page
- `privacy.html` - privacy policy
- `terms.html` - terms page
- `assets/` - logo and website assets

## Hosting Notes

This is a static HTML/CSS website. It can be hosted on Netlify, Cloudflare Pages, Vercel, GitHub Pages, or any standard web host.

For Netlify, upload this folder as the site root. The homepage is `index.html`.
